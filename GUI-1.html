Graphical User Interfacting-1(GUI-1)

Course Description:

This is a first course in the design and implementation of graphical user interfaces (GUIs) for web-based environments. The course requires the completion of several client-side programming projects that are evaluated on design and layout of the user interface, coding style, and comprehensiveness of documentation. Students learn to create web pages using HTML, CSS, JavaScript, and jQuery. The course may be taken on its own, but is intended to be followed by COMP 4620 to complete a two-course CS project sequence.
